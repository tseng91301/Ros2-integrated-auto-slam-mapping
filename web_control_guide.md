# 🌐 Web 網頁控制與實時建圖操作指南

本專案整合了 **網頁遙控器與實時地圖圖傳系統 (`wheeltec_web_teleop` 功能包)**。該系統提供專為手機與電腦瀏覽器設計的精美遙控介面，支援類空拍機雙搖桿遙控、實時 SLAM 地圖繪製、機器人定位信標、動態比例尺、以及自主探索控制與地圖直接下載。

## 🚀 1. 快速啟動網頁服務

您可以透過 Tmux 腳本一鍵啟動所有相關節點（包含底盤、雷達、建圖與網頁服務）：

```bash
cd /home/ubuntu/workspaces/isaac_ros-dev

# 啟動實體機器人自主探索建圖 (預設會開啟網頁伺服器)
./ros2_tmux_launcher.sh explore
```
*網頁伺服器會自動在機器人後端執行，監聽在連接埠 **`8080`**。*

## 📡 2. 如何連線與開啟 Web 介面

1. **取得機器人 IP 位址**：
   在機器人 Host 本機端執行 `hostname -I`，例如取得 `192.168.1.108`。
2. **開啟瀏覽器**：
   確保您的手機或電腦與機器人連接在**同一個 Wi-Fi 區域網路**。開啟瀏覽器並輸入網址：
   ```text
   http://<機器人_IP>:8080
   # 例如: http://192.168.1.108:8080
   ```
3. **入口門戶 (Portal Menu)**：
   開啟後，您將看到 **THOR ROBOTICS 門戶選單**：
   - **Manual Remote (手動控制)**：進入 `/teleop` 路徑，進行虛擬搖桿與鍵盤遙控。
   - **Auto Explorer (自主探索)**：進入 `/explorer` 路徑，進入自主建圖控制中心。

## 🕹️ 3. 手動控制頁面 (`/teleop`) 操作說明

此頁面專為手動遙控、底盤偵錯設計，採用對稱的雙控制區版面。

### 搖桿與方向鍵操作
- **左側控制區 (線性速度 - 前進/後退/側移)**：
  - 向上/下：控制機器人前進與後退。
  - 向左/右 (需開啟 **Holonomic Mode**)：控制麥克納姆輪向左或向右側向橫移 (Strafing)。
- **右側控制區 (角速度 - 自轉轉向)**：
  - 向左/右：控制機器人原地逆時針/順時針自轉。
- **控制模式切換**：
  - 左右控制區上方可切換 `JOYSTICK` (虛擬類比搖桿，推越遠速度越快) 與 `D-PAD` (九宮格按鈕，按住移動、放開即停)。

### 參數設定與安全防護
- **速度限制滑桿 (LINEAR / ANGULAR Speed Limits)**：
  - 位於網頁左下方，可即時調整輸出最大速限，確保在狹窄空間的安全。
- **Enable Holonomic Mode (Mecanum Strafing) 勾選框**：
  - 勾選後，左側搖桿/按鈕將支援全向側移。未勾選時，左側只支援前後移動。
- **EMERGENCY STOP (紅色緊急停止鈕)**：
  - 位於右下方，點擊會立即下發 `0` 速度指令，地圖視窗會觸發紅光震動警告。
- **電腦鍵盤操作支援**：
  - 電腦瀏覽器支援鍵盤控制：
    - **`i` / `ArrowUp`**：前進 | **`,` / `ArrowDown`**：後退。
    - **`j` / `ArrowLeft`**：左轉 | **`l` / `ArrowRight`**：右轉。
    - **`k` / `Space`**：即時制動停止。
    - **按住 `Shift` 鍵**：臨時進入全向側移模式。

### 手機網頁滑動防誤觸
- 在手機上使用時，網頁左側邊緣會生成細長的 **青霓虹色滾動條**。請透過此區域滑動網頁，避免在拖曳網頁時誤觸虛擬搖桿或地圖。

## 🧭 4. 自主探索頁面 (`/explorer`) 操作說明

此頁面整合了自主探索控制面板、即時遙測 HUD 以及互動式建圖畫面。

### 📊 即時遙測 HUD 資訊欄
- **STATUS (狀態)**：`PAUSED` (安全暫停/初始狀態)、`EXPLORING` (探索中)、`COMPLETE` (探索完成)。
- **ACTIVE GOAL (目標坐標)**：當前規劃前進的邊界世界座標 `(X, Y)`。
- **LAP PROGRESS (探索圈數)**：當前圈數進度 (例如 `1 / 2`)。
- **FRONTIERS (邊界數)**：當前地圖上偵測到的候選未知區域群集數量。

### 🗺️ 實時互動地圖 Canvas
- **地圖互動**：
  - 使用滑鼠滾輪或手機雙指捏合進行地圖**縮放**。
  - 按住滑鼠左鍵或單指拖曳進行地圖**平移**。
  - 地圖右上角設有快捷按鈕：`+` 放大、`-` 縮小、`⟲` 重設並置中視角。
- **圖示與標記說明**：
  - **機器人信標**：紅色實心圓，外圈有紅色雷達波紋脈動，中心白色箭頭代表朝向 (Yaw)。
  - **邊界質心 (Frontier Centroids)**：**綠色圓點**，代表演算法偵測到未知區域邊界的候選點。
  - **當前目標 (Active Target)**：**藍色瞄準十字標記**，代表機器人當下導航的目標。
- **地圖色彩定義**：
  - 🔵 **石板藍灰 (Unknown Area)**：雷達尚未探測到的未知空間。
  - ⚫ **深藍近黑 (Free Space)**：雷達已掃描、確認無障礙物之安全可行走區域。
  - 🟢 **霓虹青藍 (Occupied Area)**：探測到的障礙物與牆面邊界。

### ⚙️ 自主探索控制器功能 (Explorer Controller)
1. **START EXPLORATION**：解除安全暫停，使機器人開始運算並移動。
2. **PAUSE**：立即暫停機器人移動，並鎖定目前目標。
3. **RESUME**：繼續探索。
4. **RESET EXPLORATOR**：清除探索狀態與已儲存的黑名單，重新對全圖進行規劃。
5. **LOOP SETTINGS (圈數設定)**：
   - 設定探索最大圈數（預設為 1）。若設定為 2 或 3，第一輪探索結束後會清除黑名單，重新前往殘留的細小死角。此操作會**強制 SLAM 進行閉環檢測 (Loop Closure)**，能大幅修復並優化建圖的閉合度。

### 💾 地圖管理功能按鈕 (Map Management)
- **SAVE MAP (ROBOT)**：將當前 SLAM 地圖儲存在**機器人本機磁碟**的 `/home/ubuntu/maps/` 目錄中，產生 `map.yaml` 與 `map.pgm`。
- **RESET SLAM MAP**：呼叫服務清除 SLAM Toolbox 的記憶，重置地圖為空白。
- **DOWNLOAD PNG**：將瀏覽器畫布上的實時彩色地圖匯出並下載為 `robot_map.png`。
- **DOWNLOAD ZIP**：後端即時執行 `map_saver`，將標準 ROS 導航格式檔案（`map.yaml`, `map.pgm`, `map.png`）打包為 ZIP 壓縮檔直接下載至您的操作裝置，可用於後續的導航部署。
